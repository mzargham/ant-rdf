# BlackBox _(Class)_

[← Home](Home)

**IRI:** `https://w3id.org/ant#BlackBox`

**Label:** Black box

## Definition

A network so successfully stabilized that it is treated as a single actant; its internal complexity is invisible. Latour 2005: simplification or 'black boxing' is a necessary part of agency. Linked back to the network it punctualizes via ant:isPunctualizationOf.

## Source

Latour, B. (2005). Reassembling the Social; Law, J. (2008).
