# Durability _(Class)_

[← Home](../Home)

**IRI:** `https://w3id.org/ant#Durability`

**Label:** Durability

## Definition

An abstract category for *how* a network configuration holds. Three subclasses per Law 2008: MaterialDurability, StrategicDurability, DiscursiveStability.

## Source

Law, J. (2008). Actor Network Theory and Material Semiotics.
